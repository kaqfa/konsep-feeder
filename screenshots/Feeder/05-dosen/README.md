# 05-dosen

Daftar dosen aktif. Menampilkan NIDN, NUPTK, nama, jenis kelamin, agama, dan status.

Detail: Informasi lengkap dosen dengan tab: Detail Data, Penugasan, Aktivitas Mengajar, Riwayat Fungsional, Kepangkatan, Pendidikan, Sertifikasi, Penelitian, Pembimbing, Penguji + sub-tab Biodata/Keluarga/Kebutuhan Khusus.
